Procedure to Launch the Lockdown Browser while the Jetty is running
==================================================================
1) Install the Jetty and LDB 
2) start Jetty using the following command on the Terminal
   java -jar start.jar <configfile>&

   Note that an "&" is added at the end of the command. For example java -jar start.jar etc/admin.xml etc/demo.xml&
 
3) Press "return" to see the prompt  
4) Type "exit" at the prompt, Press "return"  and then "Quit the Terminal".  
5) Launch the LDB using "LDB_MAC\ObjectCode\LdB Launcher\build\LdB Launcher.build\Release\LdB_Launcher.build\JavaClasses\LdB_Launcher.class
6) Lockdown browser would be lauched.
